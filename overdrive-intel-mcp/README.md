# Intel Overdrive MCP

**Real-time intelligence for Claude Code.** New MCP servers, tools, skills, frameworks, and SDK updates from 1,000+ monitored sources — delivered straight to your agent.

## Install

Paste into Claude Code:

```
bash <(curl -s https://inteloverdrive.com/dl/setup.sh)
```

Or install directly:

```
npm i -g intel-overdrive-mcp
```

## What it does

Your agent's training data is months old. Intel Overdrive monitors 1,000+ sources (22k+ GitHub repos, 280+ RSS feeds, npm, PyPI, Reddit, HN, and more) and makes the latest developments searchable through a single MCP tool call.

- **Ask "what MCP servers exist for auth?"** → ranked results with star counts
- **Ask "what's new this week?"** → curated feed of recent updates
- **Your agent calls it automatically** when you ask about tools or SDKs

One API call. Instant results. No web scraping.
